🚀 Conversor de Bases Numéricas Interativo

Transforma números entre Binário (base 2), Decimal (base 10), Octal (base 8) e Sênario (base 6) com interface moderna e colorida!

✨ **Funcionalidades:**
- ✅ Conversão bidirecional completa (qualquer base → qualquer base)
- 🎨 Design responsivo com gradientes e animações suaves
- ⚡ Cálculo automático ao trocar bases ou Enter
- 💡 Exemplos clicáveis para testes rápidos
- 📱 100% mobile-friendly
- 🎯 Validação inteligente de dígitos por base

🔮 **Como usar:**
1. Selecione base origem → Digite número → Escolha destino
2. Clique "Converter Agora" ou aperte Enter

📊 **Exemplos:**
- 27 (decimal) → 11011 (binário)
- 1010 (binário) → 10 (decimal) 
- 15 (decimal) → 23 (base 6)

💻 **Tech Stack:** HTML5 | CSS3 (Flexbox/Grid) | Vanilla JavaScript
